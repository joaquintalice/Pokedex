# Execute in development

1. Clone the repository
2. Execute
```
yarn install
```
3. Have installed Nest CLI
```
npm i -g @nestjs/cli
```

4. Raise the DB
```
docker-compose up -d
```


## Stack used
* MongoDB
* Nest
